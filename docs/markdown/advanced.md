## 转义字符
- 语法：显示转义字符（如#*）,可以使用反斜杠 `\` 来进行转义。
支持以下这些符号的转义:
```
\   反斜线
`   反引号
*   星号
_   底线
{}  花括号
[]  方括号
()  括弧
#   井字号
+   加号
-   减号
.   英文句点
!   惊叹号
```
- 示例：
```
\*\* 正常显示星号 \*\*
```
- 效果对比：
    转义前： **文本加粗**
    转义后： \*\* 正常显示星号 \*\*
